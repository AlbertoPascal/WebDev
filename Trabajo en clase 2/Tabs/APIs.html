Código del API de Amazon

El siguiente código incorpora la búsqueda de amazon para buscar los libros de la serie Hunger Games.

La operación de ItemSearch Es la encargada de realizar la búsqueda.

http://webservices.amazon.com/onca/xml?
Service=AWSECommerceService&
AWSAccessKeyId=[AWS Access Key ID]&
AssociateTag=[Associate ID]&
Operation=ItemSearch&
Keywords=the%20hunger%20games&
SearchIndex=Books
&Timestamp=[YYYY-MM-DDThh:mm:ssZ]
&Signature=[Request Signature]
