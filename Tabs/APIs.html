Sobre Nosotros

Nuestro proyecto consistirá en una plataforma web para organizar finanzas personales. La plataforma será una que ayude a las personas a organizar y administrar sus finanzas personales. En esta página, cualquier persona podrá registrarse y llevar un registro de sus ganancias mensuales. En base a esto, la página deducirá la cantidad de dinero que puede gastar y el usuario podrá buscar objetos que quiera comprar, para que la página le recomiende las mejores opciones de compra. Adicionalmente, las recomendaciones de la página se basarán en los mejores precios disponibles (o un promedio de éstos) para una lista de artículos dada y se calculará la mejor combinación de cantidad de objetos dentro del rango de gastos que tienes “permitidos”. Finalmente, la página te recomendará los lugares más populares donde puedes obtener esos artículos.

Descripcion del Sistema

El sistema deberá permitir facilitar al usuario una organización de su budget personal. Es decir, al ingresar el sueldo con el que se cuenta y la lista de objetos que se desean adquirir, la plataforma calculará la cantidad aproximada que la persona podría gastar y realizará búsquedas para tratar de obtener la mayor cantidad de objetos de tu lista alcanzables con el presupuesto disponible. Alternativamente, se puede realizar un cálculo aproximado como plan de ahorros para poder comprar dichos objetos. La plataforma se apoyará de una base de datos en conjunto con el API de pricing de amazon y el api de google maps para proveer de búsquedas de dónde comprar la mayor cantidad de objetos deseados según tu presupuesto y proporcionar la ubicación de las tiendas cercanas a ti que vendan dichos objetos. Por ende, los precios serán únicamente de referencia.

Problematica a Resolver

Cita

La gente tiende a sobreestimar sus ingresos y a subestimar sus deudas, lo que hace que la mayoría de las veces haya desequilibrios entre lo que gasta y lo que gana
-Luis Fabre vicepresidente técnico de la Condusef

La mayoría de los mexicanos, sobretodo aquellos en condiciones de pobreza, desconocen el cómo manejar sus fondos. Debido a esto, es común que las personas gasten de más o no puedan alcanzar a comprar todos los artículos que necesitan o que desean adquirir. Adicionalmente, a falta de organización financiera, las personas se ven en situaciones complicadas al tratar de ahorrar para comprar objetos de mayor precio. Por ende, es necesario brindar recomendaciones sobre cómo ahorrar para lograr obtener un producto en un tiempo razonable.

Código del API de Amazon

El siguiente código incorpora la búsqueda de amazon para buscar los libros de la serie Hunger Games.

La operación de ItemSearch Es la encargada de realizar la búsqueda.
http://webservices.amazon.com/onca/xml?
Service=AWSECommerceService&
AWSAccessKeyId=[AWS Access Key ID]&
AssociateTag=[Associate ID]&
Operation=ItemSearch&
Keywords=the%20hunger%20games&
SearchIndex=Books
&Timestamp=[YYYY-MM-DDThh:mm:ssZ]
&Signature=[Request Signature]
